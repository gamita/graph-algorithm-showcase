Minimum Spanning Tree: Prim Algorithm

Wikipedia Link ↗
This demo visualizes a process of Prim Algorithm (Please click Step 1 "Graph" button, then click Step2 "Run" button). Prim Algorithm finds a minimum spanning tree (MST) from given graph. Minimum spanning tree is a tree which connects among all nodes with minimum sum of edge weight. This demo shows edge choice process of minimum spanning tree step by step from the following graph. Finally, the set of orange edges is a minimum spanning tree of the graph, which means that all nodes are connected by orange edges with minimum total length.
